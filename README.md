# User Management Web Application

This project is a simple Full Stack Web Application developed using **Python (Flask), HTML, CSS, and SQLite**.  
It demonstrates how frontend, backend, and database components work together to build a complete web application.



## Features
- Add new user details through a web form
- Store user data in SQLite database
- View all users in a table
- Simple and clean user interface



## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
